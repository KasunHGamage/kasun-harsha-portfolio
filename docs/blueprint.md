# **App Name**: Kasun Harsha Portfolio

## Core Features:

- Sticky Navigation Header: Implement a sticky header with animated active section indicator and smooth scroll navigation.
- Animated Hero Section: Create a hero section with an animated role text switcher and subtle floating/motion effects.
- Interactive Project Cards: Design and implement interactive project cards with hover animations to showcase featured projects.
- ReactBits-Inspired Skills Display: Develop skill cards or tabs using ReactBits UI patterns to categorize and display skills.
- Clean Contact Form: Create a clean contact form with floating labels and button loading/success state animations.
- Framer Motion Animations: Incorporate subtle Framer Motion animations for section entrances, hover interactions, and the navigation indicator to create a polished user experience.
- Fully Responsive Layout: Ensure the website is fully responsive across various devices using a mobile-first approach and Tailwind CSS.

## Style Guidelines:

- Primary color: Dark indigo (#4F46E5) for a modern and professional look.
- Background color: Very dark gray (#1A202C) for a dark, modern theme.
- Accent color: Light blue (#90CDF4) to highlight key elements and CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look; suitable for both headlines and body text
- Use simple, minimalist icons for navigation and social links.
- Implement a clean, developer-focused aesthetic with clear visual hierarchy and spacing, inspired by ReactBits UI patterns.
- Use subtle animations with Framer Motion for section entrances, hover interactions, and navigation indicator.
